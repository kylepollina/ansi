# ansi.v

Simple module for the V programming language for formatting ANSI escape sequences to the terminal.

![img](example.png)

--------

#### Styles
- reset
- bold
- dim
- underline
- blink
- reverse
- hidden

#### Foreground Colours
- black
- red
- green
- yellow
- blue
- magenta
- cyan
- white

#### Background Colours
- on_black
- on_red
- on_green
- on_yellow
- on_blue
- on_magenta
- on_cyan
- on_white
