# colo.v

Colors in your terminal for the V programming language

![example](exaple.png)

Works on MacOS, Linux, Windows

--------

#### Styles
- `colo.reset`
- `colo.bold`
- `colo.dim`
- `colo.underline`
- `colo.blink`
- `colo.reverse`
- `colo.hidden`
