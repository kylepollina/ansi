# ansi.v

Colors in your terminal for the V programming language

![img](example.png)

Works on MacOS, Linux, Windows

--------

#### Styles
- reset
- bold
- dim
- underline
- blink
- reverse
- hidden

#### Foreground Colours
- black
- red
- green
- yellow
- blue
- magenta
- cyan
- white

#### Background Colours
- on_black
- on_red
- on_green
- on_yellow
- on_blue
- on_magenta
- on_cyan
- on_white
